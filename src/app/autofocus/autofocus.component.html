Autofocus

Import AutoFocusModule into your app.module.ts:


    {{"import \{ AutoFocusModule \} from 'nw-style-guide/autofocus';

    ...........
    ...........

    @NgModule({
        declarations: [...],
        imports: [
            .....
            .....
            AutoFocusModule
        ],
        bootstrap: [AppComponent]
    })
    export class AppModule \{ \}"}}

Then use directive nwAutofocus in template by adding it to the element you'd like autofocused on load.
