Append dropdown to body

To append the dropdown menu to the body, the recommended approach is to leverage the popover module from ngx-bootstrap. It is also recommended to use the containerClass with at least popover-dropdown. This class will remove a lot of base popver styling.

Dropdown backdrop

To render a backdrop to catch background clicks insert an element with the class dropdown-backdrop as the first child of your .dropdown element.

Backdrops should not be used if autoClose is set to false
