Toasts

This feature is currently styles only. The onus is on the developer to include the animations below as well as any logic required for showing / hiding / dismissing and queuing notifications This should be converted to an Angular module to include functionality and logic.

{{toast.message}}

Implementation details

  • Toast messages should ideally be limited to no more than 50 character
  • Successful toast messages should not be manually dismissable
  • The default time to display the toast is 3 seconds
